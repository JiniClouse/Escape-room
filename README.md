## על הפרויקט

"הגנוז בקרקוב" הוא חדר בריחה דיגיטלי מלא הפועל ישירות בדפדפן, ללא התקנה.
השחקניות מתמודדות עם שרשרת של חידות הסמויות בתוך עולם ויזואלי אטמוספרי,
עם טיימר לחץ של 30 דקות ומוזיקת רקע מרגשת.

### שלבי המשחק
1. פתיחת הפורטרט הסגור
2. פיצוח קוד הכניסה
3. חידת ספריית הספרים
4. פתיחת הספרייה הסודית
5. פתיחת התיבה הנעולה
6. גילוי הסוד הגנוז

---

## הפעלה

**אונליין:** לחצו על הקישור למעלה מכל דפדפן מודרני.

**מקומית:**
```
1. הורידו את כל הקבצים (Code → Download ZIP)
2. פתחו את index.html בדפדפן
3. וודאו שתיקיית assets נמצאת באותה תיקייה
```

> מומלץ: Chrome / Edge במסך מלא (F11)
